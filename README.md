# Elemental Monsters

A simple monster battler using a 5 element rock, paper, scissor inspired system.  The element count is configurable based upon the elements listed in dmgmodifiers.csv.  The elements currently in use are:

- Water
- Fire
- Earth
- Wind
- Death
